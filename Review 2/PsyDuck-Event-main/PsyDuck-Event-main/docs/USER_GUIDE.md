# 📖 User Guide

This guide explains how to interact with the PsyDuck Events platform as either a Student or an Administrator.

## 👤 Student Experience

### 1. Discovering Events
- On the **Home Page**, you can see featured events in the high-impact hero section.
- Click **"Browse"** in the top navigation to see all available campus events in a clean grid.
- Use the **Search Bar** to filter events by title or category.

### 2. Viewing Details
- Click on any event card to open the **Editorial Detail View**.
- **Tabs**: Switch between **Overview** (Details), **Timeline** (Schedule), and **Organizer** (Contact Info).
- The **Timeline** shows the exact sequence of activities with bold time markers.

### 3. Registration
- Locate the **Attendance Card** on the right side of the event details.
- If seats are available, click **"Confirm Spot"**.
- Once registered, a green **"Verified"** badge will appear.
- If you change your mind, click **"Cancel Attendance"** to free up the spot for other students.

---

## 🛡️ Admin Experience

### 1. Accessing the Dashboard
- Navigate to `/admin`.
- Log in with your admin credentials.
- The dashboard provides a high-level view of all events, current registration counts, and percentage of attendance.

### 2. Managing Events
- **Create**: Click **"Create Event"** to open the editor.
- **Edit**: Click the **"Edit"** button next to any event in the table.
- **Fields**:
    - **Schedule**: Enter items separated by a semicolon (e.g., `09:00 AM - Intro; 10:00 AM - Keynote`).
    - **Plans**: Detailed description for the Overview tab.
    - **Contact**: Emails or phone numbers for the Organizer tab.

### 3. Exporting Data
- Click the **"Export"** button in the top right of the Admin Dashboard.
- A CSV file (`events_export.csv`) will be downloaded, containing all current event metrics.
